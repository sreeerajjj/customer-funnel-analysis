Customer Funnel Analysis & Conversion Dashboard

Overview
This project analyzes user behavior across different stages of the customer journey (Visit → Product View → Cart → Purchase) to identify conversion rates and drop-off points.

Objective
- Track user movement through funnel stages  
- Identify where users drop off  
- Analyze conversion performance  
- Perform segmentation based on user attributes  

Tools & Technologies
- SQL  
- Power BI  
- DAX  
- Excel (for data handling)  


Dataset
- User interaction dataset (sample included in this repository)  
- Contains user events such as visits, product views, cart actions, and purchases  


Process
1. Data Cleaning
- Removed null values  
- Standardized formats  
- Prepared structured dataset  

 2. Funnel Creation
- Created user-level funnel stages:
  - Visited  
  - Viewed Product  
  - Added to Cart  
  - Purchased  

3. Data Transformation
- Aggregated data at user level  
- Avoided duplication issues  
- Created final dataset for analysis  

4. Dashboard Development
- Built interactive Power BI dashboard  
- Created KPIs and DAX measures  
- Visualized funnel and drop-offs  



Key Insights
- Significant drop-off observed between cart and purchase stage  
- Organic traffic shows higher conversion compared to other sources  
- Mobile users have lower conversion rates than desktop users
